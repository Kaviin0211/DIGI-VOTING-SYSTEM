# DIGI-VOTING-SYSTEM
**Digi Voting** is a web-based online election system built using **HTML, CSS, and JavaScript** in **VS Code**. It has separate **Admin** and **User** pages. Users verify their Voter ID to vote, while Admins manage voters, view results, and check winners. The system ensures secure, transparent, and efficient digital voting.
# 🗳️ Digital Voting System

A secure and user-friendly Online Digital Voting System developed using HTML, CSS, JavaScript, and MySQL to simplify online polling, voting, and result management with real-time processing and secure authentication.

---

# 🚀 Project Overview

The Digital Voting System is a web-based application designed to modernize the traditional voting and polling process.

The system allows users to:
- Participate in online voting securely
- Access polls remotely
- View real-time results
- Experience fast and efficient voting

This project provides an easy-to-use interface for both administrators and voters while ensuring transparency, fairness, and efficiency.
